Detailed Description

Declarations of functions that deal with the right hand side of the ODEs.

Authors:
JR, modified by Yoginho, Manu, Damjan and Anton

Copyright (C) 1989-2003 John Reinitz, 2009-2013 Damjan Cicin-Sain, Anton Crombach and Yogi Jaeger

We have an initializing function called InitZygote() that reads parameters, defs data and search space limits from a data file and installs a couple of things like the solver and temporary arrays for the dvdt function. Then we have dvdt_orig, which is the inner loop of the model. It propagates the equations from one time point to another and gets called whenever Blastoderm is in PROPAGATE mode. Last but not least, we have a few mutator functions in this file. They change the local lparm EqParm struct.
