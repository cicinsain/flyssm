ERROR HANDLING FUNCTIONS The two routines 'error' and 'warning' print error messages with value. 'error'then exits, while 'warning' returns to the calling function.

Both functions are called analogous to the way you would call printf. The only conversion specs that are legal are c, d, f, g and s. No modifiers (h,l or float precision or size specifiers are allowed).
