\subsection{Detailed Description}
Defines a lot of constants, bias-\/related stuff and functions to deal with the data structures. \begin{DoxyAuthor}{Author}
JR, modified by Yoginho
\end{DoxyAuthor}
Copyright (C) 1989-\/2003 John Reinitz, 2009-\/2013 Damjan Cicin-\/Sain, Anton Crombach and Yogi Jaeger

1. structs and constants for things used throughout the fly model specific part of the code (since this file will al-\/ ways get included when we run the fly model); therefore all fly-\/specific things that used to be in \hyperlink{global_8h}{global.h} are now in here 2. stuff that deals with that part of the blastoderm which is fixed by the maternal genotype, i.e. division schedules (including the rules for \hyperlink{zygotic_8c}{zygotic.c}, the number of nuclei at each cleavage cycle and the lineage number of the most anterior nucleus for each cleavage cycle), bicoid gra-\/ dients and diffusion schedules 3. bias-\/related stuff is also in here since it's needed to run the model
